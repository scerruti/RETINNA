# RETINNA PA3 Presentation Creation Task

## Objective
Create two Slidev presentations for the RETINNA PA3 project:
1. **10-minute version** — Technical pitch, 5-6 slides, <50 words per slide, learning objectives focused
2. **Extended version** — 25-30 minute talk, ~10 slides, deeper technical detail, FCN vs U-Net comparison

## Context
- **Project**: Wildfire burn scar detection using Sentinel-2 satellite imagery
- **Learning Objectives**: FCN architecture, training pipeline, dataset preprocessing, validation metrics, hyperparameter tuning, class imbalance handling, architectural comparison
- **Special Focus**: False negatives in validation (model biases toward predicting "unburned")
- **Visualizations**: Pixel-level heatmaps showing false negatives + metrics tables (IoU, precision, recall, false negative rate)
- **Tool**: cc-slidev plugin (already installed)

## Step-by-Step Process

### Phase 1: Create 10-Minute Plan
1. Read `/Users/scerruti/RETINNA/docs/PA3_SUMMARY.md` to understand learning objectives
2. Design 10-minute structure:
   - Slide 1: Context & motivation (PA3 to RETINNA adaptation)
   - Slide 2: Architecture template (FCN encoder/decoder, 3→11 channels, 27→2 classes)
   - Slide 3: U-Net vs FCN comparison (visual diagram)
   - Slide 4: Loss function & training strategy (BCE + Dice for class imbalance)
   - Slide 5: Validation results & false negatives (pixel heatmap + metrics table)
   - Slide 6: Learning objectives achieved (checklist)

### Phase 2: Create Extended Plan
1. Design 25-30 minute structure (~10 slides):
   - Slides 1-2: Backstory & academic context (PA3 assignment, RETINNA adaptation)
   - Slides 3-4: Data description (Sentinel-2, normalization, one-hot encoding)
   - Slides 5-6: FCN & U-Net architecture deep dives (batch norm, skip connections)
   - Slides 7-8: Training setup & initial results (false negatives problem identified)
   - Slides 9-10: Hyperparameter tuning strategy & outcomes (batch size, learning rate, loss weighting)
   - Slides 11-12: FCN vs U-Net empirical comparison (metrics, inference speed, boundary preservation)
   - Slides 13-14: Learning objectives achieved (detailed checklist)

### Phase 3: Generate Slides
1. Use `/slidev:plan` for 10-minute version
   - Interactively refine the plan structure
   - Ensure <50 words per slide, ≤6 elements
   - Approve the plan
2. Use `/slidev:from-plan` to generate 10-minute slides
3. Use `/slidev:plan` for extended version
   - Allow more depth and complexity
   - Approve the plan
4. Use `/slidev:from-plan` to generate extended slides

### Phase 4: Visualizations
1. Identify validation results from `/Users/scerruti/RETINNA/notebooks/03_training.ipynb`
2. For false negatives visualization:
   - Extract pixel-level predictions and ground truth
   - Create heatmap highlighting false negatives (predicted unburned, actual burned)
   - Generate metrics table with IoU, precision, recall, false negative rate
3. Include visualizations in both presentations

### Phase 5: Completion
- Export both presentations to PDF
- Verify all slides are present and readable
